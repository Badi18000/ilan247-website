# Design Context — ilan247

## Design Context

### Users
Propriétaires de PME et commerçants de la région de Montréal qui ont besoin de supports publicitaires (roll-up, enseignes, impressions grand format). Décideurs rapides qui cherchent un prestataire fiable, disponible et qui livrera un résultat professionnel dans les délais. Contexte d'usage : souvent depuis un téléphone, en recherche active d'un fournisseur.

### Brand Personality
**Professionnel & Fiable** — Artisan expert, digne de confiance, réactif 24/7.
Trois mots : **Compétent. Accessible. Énergique.**
Ton : chaleureux et direct, jamais corporate ou froid. Montréal local.

### Aesthetic Direction
- **Référence** : Vistaprint / Canva — moderne, accessible, coloré, approchable. Design plat avec accents forts.
- **Palette** : Orange vif (brand) + vert émeraude (accent) sur fonds blancs/slate clairs.
- **Typographie** : Rajdhani pour les titres (impact, moderne), IBM Plex Sans pour le corps (lisible, confiant).
- **Anti-référence** : Pas d'esthétique "agence créative branchée" (trop froid), pas de glassmorphism, pas de dark mode avec néons.
- **Thème** : Light mode. Énergique mais professionnel. Couleurs vives qui évoquent l'impression physique — l'orange de l'encre, le vert de la végétation montréalaise.

### Design Principles
1. **Confiance avant tout** : Chaque interaction doit renforcer la crédibilité — animations fluides, états clairs, jamais de confusion.
2. **Énergie sans chaos** : Le mouvement amplifie le contenu, ne le distrait pas. Animations courtes (< 0.7s), easing naturel.
3. **Local et humain** : Copie chaleureuse, états de succès célébrés comme une vraie victoire pour le client.
4. **Delight discret** : Les moments joyeux se découvrent, ils ne s'imposent pas. Pas de confettis au chargement — seulement quand l'action mérite une célébration.
5. **Mobile d'abord** : Les clients cherchent sur téléphone. Chaque touch target, chaque animation doit fonctionner sur mobile.
